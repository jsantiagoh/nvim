# nvim
Nvim repository

## Installation

```
❯ mkdir -p ~/.config
❯ cd ~/.config
❯ git clone --recursive https://github.com/jsantiagoh/nvim.git 
```

## First run and packages

This setup uses [minpac](https://github.com/k-takata/minpac) , after installation run inside nvim


```
:call minpac#update()
```
